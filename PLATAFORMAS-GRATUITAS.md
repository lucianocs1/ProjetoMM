## 🔄 **PROCESSO AUTOMATIZADO**

### **1. Executar Script:**
```batch
deploy-producao.bat
```

### **2. Deploy Sequencial:**
1. **Railway:** Backend (5 min)
2. **Vercel:** Admin (3 min)  
3. **Vercel:** Frontend (3 min)

### **3. Configurar URLs:**
1. Anotar URL do Railway
2. Atualizar apiConfig.js (admin)
3. Atualizar api.js (frontend)
4. Commit → Deploy automático

- Login admin ✅
- Cadastrar produto ✅  
- Ver produto na loja ✅
